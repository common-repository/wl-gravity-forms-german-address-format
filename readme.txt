== Description ==

Dieses Plugin ändert das Adressformat von Gravity Forms in das deutsche Format, bei dem die Postleitzahl vor dem Ort steht.

== Installation ==

Nutzen Sie die Suche im WordPress Plugin Verzeichnis und suchen nach 'WL GF German Address Format', oder laden Sie das Plugin auf der offiziellen WordPress Plugin Seite herunter und laden dies manuell in Ihre WordPress Instanz hoch.
